# ÉLAN 🍷

A premium high-end restaurant website concept focused on cinematic storytelling, luxury aesthetics, and immersive user experiences.

## Features

* Fullscreen food cinematics
* Smooth scroll storytelling
* Elegant menu transitions
* Smoke & fire hover effects
* Interactive reservation experience
* Ambient restaurant atmosphere
* Premium typography & motion design
* Award-winning inspired UI/UX

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* GSAP
* Framer Motion
* Lenis
* Three.js
* React Three Fiber

## Goal

Create a restaurant website that feels like a cinematic dining experience rather than a traditional menu website.

> "A restaurant website should make users feel the atmosphere before they even step inside."
